Проєкт 1: Веб-додаток «Інтерактивна модель вітряків»

  • Опис: Це логічна гра, створена для демонстрації взаємодії об'єктів та обробки станів у реальному часі. Користувач може вмикати або вимикати три вітряки. Система аналізує кількість активних елементів та активує відповідні сценарії:
    • 1 активний вітряк: система перебуває в режимі очікування (візуальні зміни відсутні).
    • 2 активні вітряки: активується лампа (світловий індикатор).
    • 3 активні вітряки: спрацьовує кумулятивний ефект — вмикається лампа та запускається рух автомобіля.
  • Технології: HTML5 (структура), CSS3 (стилізація та анімація), JavaScript (логіка станів), Git (контроль версій).
